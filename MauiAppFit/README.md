# MauiAppFit

## Sobre o Projeto

O MauiAppFit é uma aplicação voltada para o registro e acompanhamento de atividades físicas. O usuário pode cadastrar atividades, informar dados relacionados ao exercício realizado e consultar seu histórico de registros.

## Funcionalidades

Cadastro de atividades físicas.
Registro de peso e observações.
Listagem das atividades cadastradas.
Organização dos dados em diferentes telas.

## Conteúdos Aprendidos

Durante o desenvolvimento deste projeto foram estudados os seguintes conceitos:

Arquitetura MVVM (Model-View-ViewModel).
Separação de responsabilidades entre interface, lógica e dados.
Utilização do BindingContext.
Data Binding.
Atualização automática da interface gráfica.
Organização de projetos de médio porte.

## Destaque

O padrão MVVM tornou o código mais organizado e de fácil manutenção. A View ficou responsável apenas pela interface gráfica, enquanto o ViewModel concentrou a lógica da aplicação e a comunicação com os dados.

O BindingContext foi um dos conceitos mais importantes do projeto, pois permitiu conectar propriedades do ViewModel aos elementos visuais da interface. Dessa forma, qualquer alteração nos dados é refletida automaticamente na tela, reduzindo a quantidade de código necessário para atualizar a interface e tornando a aplicação mais desacoplada e escalável.
