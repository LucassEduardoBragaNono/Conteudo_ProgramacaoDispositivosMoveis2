# AppJogoDaVelha

## Sobre o Projeto

O AppJogoDaVelha é uma implementação digital do clássico jogo da velha desenvolvida em .NET MAUI. Dois jogadores podem interagir com o tabuleiro, realizando jogadas alternadas até que haja um vencedor ou empate.

## Funcionalidades

Tabuleiro interativo.
Alternância automática entre os jogadores.
Verificação de vitória.
Verificação de empate.
Reinício da partida.

## Conteúdos Aprendidos

Durante o desenvolvimento deste projeto foram estudados os seguintes conceitos:

Utilização do componente Grid para construção do tabuleiro.
Organização de elementos visuais em linhas e colunas.
Manipulação de eventos Clicked.
Captura de eventos definidos no XAML pelo código C# (Code Behind).
Lógica de programação aplicada ao controle do jogo.
Atualização dinâmica da interface gráfica.
