# MauiAppHotel

## Sobre o Projeto

O MauiAppHotel é uma aplicação de reservas de hospedagem. O usuário informa os dados necessários para a reserva e o sistema apresenta as informações da estadia de forma organizada.

## Funcionalidades

Cadastro de informações da reserva.
Exibição dos dados informados.
Navegação entre telas.
Apresentação do resumo da hospedagem.


## Conteúdos Aprendidos

Durante o desenvolvimento deste projeto foram estudados os seguintes conceitos:

Utilização do padrão BindingContext.
Associação de dados entre objetos e elementos visuais.
Transferência de informações entre páginas.
Criação de modelos de dados.
Navegação entre telas.
Separação entre interface e dados.

## Destaque

O conceito de BindingContext permitiu vincular propriedades de objetos diretamente aos componentes da interface gráfica, reduzindo a necessidade de atualizações manuais dos elementos visuais.
