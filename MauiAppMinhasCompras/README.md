# MauiAppMinhasCompras

## Sobre o Projeto

O MauiAppMinhasCompras é uma aplicação de gerenciamento de compras que permite cadastrar, editar, excluir, pesquisar e listar produtos armazenados localmente no dispositivo.

## Funcionalidades

Cadastro de produtos.
Edição de produtos existentes.
Exclusão de registros.
Pesquisa de produtos.
Listagem completa dos itens cadastrados.

## Conteúdos Aprendidos

Durante o desenvolvimento deste projeto foram estudados os seguintes conceitos:

Utilização do banco de dados SQLite.
Operações CRUD (Create, Read, Update e Delete).
Persistência de dados em dispositivos móveis.
Criação de classes de modelo.
Consultas ao banco de dados.
Organização de código em camadas.

## Destaque

O SQLite possibilitou armazenar os dados localmente, permitindo que as informações permanecessem salvas mesmo após o fechamento da aplicação.
