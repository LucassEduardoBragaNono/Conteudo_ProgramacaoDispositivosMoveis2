# AppImcMaui

## Sobre o Projeto

O AppImcMaui é uma aplicação desenvolvida para calcular o Índice de Massa Corporal (IMC). O usuário informa seu peso e altura, e o sistema realiza o cálculo automaticamente, apresentando o valor do IMC e sua respectiva classificação.

## Funcionalidades

Cadastro de peso e altura.
Cálculo automático do IMC.
Exibição da classificação do resultado.
Interface simples e intuitiva.

## Conteúdos Aprendidos

Durante o desenvolvimento deste projeto foram estudados os seguintes conceitos:

Entrada e validação de dados.
Operações matemáticas em C#.
Estruturas condicionais para classificação do IMC.
Manipulação de eventos de interface.
Construção de telas utilizando XAML.
Integração entre XAML e código C#.
