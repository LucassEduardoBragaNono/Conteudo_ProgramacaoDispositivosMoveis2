# AppMeuCombustivel

## Sobre o Projeto

O AppMeuCombustivel é uma aplicação desenvolvida em .NET MAUI que auxilia o usuário na escolha do combustível mais vantajoso para abastecer seu veículo. O usuário informa o preço do etanol e da gasolina, e o aplicativo realiza o cálculo automaticamente, indicando qual opção possui o melhor custo-benefício.

## Funcionalidades

Inserção do preço da gasolina.
Inserção do preço do etanol.
Cálculo automático da melhor opção de combustível.
Exibição do resultado na tela.

## Conteúdos Aprendidos

Durante o desenvolvimento deste projeto foram estudados os seguintes conceitos:

Entrada de dados através de componentes da interface gráfica.
Conversão de dados de texto para valores numéricos.
Processamento de informações utilizando estruturas condicionais.
Exibição de resultados para o usuário.
Manipulação de eventos de botões utilizando o evento Clicked.
Criação de interfaces gráficas com XAML.
