# MauiAppTempoAgora

## Sobre o Projeto

O MauiAppTempoAgora é uma aplicação que consulta informações meteorológicas em tempo real através de uma API externa e apresenta os dados ao usuário de forma simples e organizada.

## Funcionalidades

Consulta de previsão do tempo.
Busca de informações por cidade.
Exibição de temperatura, latitude e longitude.
Consumo de serviços externos via API.

## Conteúdos Aprendidos

Durante o desenvolvimento deste projeto foram estudados os seguintes conceitos:

Consumo de APIs REST.
Desserialização de dados JSON.
Programação assíncrona com async e await.
Utilização do recurso Geolocation.
Tratamento de exceções utilizando blocos try-catch.
Comunicação entre aplicação e serviços externos.

## Destaque

O tratamento adequado das exceções foi fundamental para garantir que erros de conexão, localização ou consulta não causassem falhas na aplicação.
